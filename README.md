# Sorting-Visualizer
This is a sorting algorithm visualizer made using Javascript, HTML and CSS that helps us visualize the sorting algorithms .

Implemented algorithms are:

1. Bubble sort 
2. Selection sort 
3. Insertion sort 
4. Merge sort 
5. Quick sort 
6. Heap sort
